## 📁 Repository Structure
* `app.py`: Core Streamlit web application interface.
* `predict.py`: Core execution logic and predictive inference engine.
* `stratum_q_kernel.py`: Quantum circuit mappings and quantum feature maps.
* `stratum_q_dermo_BEST.pth`: Trained weights for the dermatology diagnostic engine.
* `stratum_q_resp_BEST.pth`: Trained weights for the pulmonology diagnostic engine.
